# tm-calculator

## TPE/PLEX Calculator
Used for calculating replacement fluid orders for therapeutic plasma exchange. Total blood volume is calculated using Nadler's equation (https://pubmed.ncbi.nlm.nih.gov/21936146/). Albumin is rounded up to the nearest 12.5g (1 vial of albumin contains 12.5g). Volume for 1 unit of FFP is estimated to be 250 ml.
